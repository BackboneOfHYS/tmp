awk concept

awk 기본 내부 함수 정리

  • gsub: 전체 문자열 치환
  • sub: 첫 번째 문자열 치환
  • split: 문자열 분리 (배열 저장)
  • index: 문자열 위치 찾기
  • length: 문자열 길이 반환
  • substr: 문자열 부분 추출
  • match: 정규식 일치 여부 확인
  • tolower: 소문자 변환
  • toupper: 대문자 변환
  • sprintf: 형식 지정 문자열 생성
  • int: 정수 변환
  • sqrt: 제곱근 계산
  • rand: 난수 생성
  • systime: 시스템 시간 확인
  • strftime: 날짜 형식 변환
  • system: 외부 명령어 실행
