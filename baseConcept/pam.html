PAM concept

PAM 옵션창 진입 및 기타 오류 해결법

  • 파일 권한 확인 - stat, ll
  • 디스크 사용량 확인 - df -h
  • SElinux 활성화라면 설정 확인 - sestatus
  • 비밀번호 정책 확인 - chage -l [사용자명], chage -M [기간일수] [사용자명]
  • 루트 파티션 읽기, 쓰기로 재설정 후 마운트 - mount -o remount, rw /
  • pat-auth-updatePAM 옵션창? 진입 명령어

Pwquailty password strength checking

  • 비밀번호 복잡도 검사
  • 길이, 특수문자, 대문자, 숫자 등 규칙 강제
  • /etc/security/pwquality.conf로 세부 설정

Kerberos authentication

  • Kerberos 기반 중앙 인증(티켓 방식)
  • 비밀번호를 직접 보내지 않고 티켓으로 인증

SSS required smart card authentication

  • 스마트카드(물리 토큰) 필수 인증
  • SSSD(System Security Services Daemon) 기반

SSS optional smart card authentication

  • 스마트카드 있으면 사용, 없어도 로그인 가능

Unix authentication

  • 기본 로컬 인증 - /etc/passwd, /etc/shadow
  • 우리가 평소 쓰는 리눅스 로그인 방식

Winbind NT/Active Directory authentication

  • Windows AD 연동 (Samba 기반)
  • 도메인 계정으로 리눅스 로그인 가능

SSS authentication

  • SSSD 기반 인증 (LDAP, AD 등 통합)
  • 요즘은 Winbind보다 이걸 더 많이 씀

Register user sessions in the systemd control group hierachy

  • 로그인 세션을 systemd cgroup으로 관리
  • 리소스 추적, 세션 관리 가능

Create home directory on login

  • 첫 로그인 시 /home/username 자동 생성
  • LDAP/AD 사용자에게 유용
