Linux /etc/pam.d 정리

기본 개념

  • 로그인, 인증 방식을 프로그램 별로 정의하는 곳
  • 로그인 방식, 2FA 추가(?), 계정 정책 적용 등을 설정 파일로 제어

디렉토리 구조

  • /etc/pam.d/login
  • /etc/pam.d/sshd
  • /etc/pam.d/su
  • /etc/pam.d/sudo
  • /etc/pam.d/passwd

내부 구조

  • type control module-path arguments

type

  • auth인증
  • account계정 상태 확인
  • password비밀번호 변경정책
  • session로그인 후 처리

control

  • required실패하면 최종 실패(하지만 계속 진행)
  • requisite실패하면 즉시 중단
  • sufficient성공하면 바로 성공처리
  • optional결과 영향 거의 없음

module

  • pam_unix.so기본 로컬 인증
  • pam_env.so환경변수 설정
  • pam_limits.so리소스 제한
  • pam_securetty.soroot 로그인 제한
  • pam_tally2.so로그인 실패 제한
