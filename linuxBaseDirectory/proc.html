Linux proc directory 정리

기본 개념

  • 커널의 메모리 속에 존재하는 가상 파일 시스템
  • 시스템의 현재 상태를 파일 형태로 보여주는 역할

/proc/$PID 구조

  • /cmdline실행시 사용한 명령행 인자
  • /environ환경변수
  • /exe실행파일 심볼릭 링크
  • /fd파일 디스크립터
  • /status프로세스 현상태

/proc/ 구조

  • /cpuinfo
  • /meminfo
  • /version커널, 컴파일, 배포판 정보
  • /uptime부팅 경과시간
  • /partition
  • /loadavg시스템 평균 부하량

/proc/sys 구조

  • sysctl 명령어를 통해 커널 파라미터 실시간 관리
  • sysctl.conf는 부팅시 설정값
  • /proc/sys는 현상태 설정값
  • /net
  • /vm
  • /kernel커널, 컴파일, 배포판 정보

/proc/sys/vm/drop_caches 구조

  • cat 1 > /proc/sys/vm/drop_caches: 페이지 캐시 비우기
  • cat 2 > /proc/sys/vm/drop_caches: 디렉토리 엔트리, 아이노드 비우기
  • cat 3 > /proc/sys/vm/drop_caches: 1+2=3
