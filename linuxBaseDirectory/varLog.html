Linux /var/log directory 정리

기본 개념

  • 대부분의 로그 디렉토리의 루트
  • 로그 관리 방식은 rsyslog(syslog의 개선), systemd-journald 방식 혼용
  • rsyslog텍스트 파일 형태의 로그 관리 시스템
  • systemd-journald바이너리 파일 형태로 전용 명령어 journalctl을 통해 조회
  • logrotate오래된 로그 압축, 삭제 관리

특수 로그 방식

  • /proc/sys/kernel/printk커널 중요도에 따라 콘솔로 출력하는 기능의 기본 설정파일
  • Kernel Ring Buffer - dmesg
  • /dev/kmsg

로그 목록

  • /var/log/messageCentOS, RHEL 계열
  • /var/log/syslogUbuntu, Debian 계열
  • /var/log/auth.log로그인 시도, sudo 사용 등 인증 관련 기록
  • /var/log/dmesg커널 및 하드웨어 장치 관련 메시지
  • /var/log/secure보안 및 인증 관련 로그(주로 RedHat 계열)
  • /var/log/apache2, nginx 등웹 서버 접속 및 에러 로그

설정 파일

  • /etc/syslog.conf, rsyslog.confrsyslog(syslog)의 설정파일
  • /etc/systemd/journald.confsystemd-journald의 설정파일
