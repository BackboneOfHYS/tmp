Linux repo 정리

기본 개념

  • /etc/yum.repos.d/*.repo파일 위치
  • /etc/yum.conf전역 옵션 설정
  • vi /etc/yum.repos.d/myrepo.repo사용자 정의 repo 파일
  • yum은 baseurl을 보고 의존성 검사 등을 수행

repo 파일 포맷

  • [$value]repo 고유 이름
  • name=$value별칭
  • baseurl=$valuerepo 서버 및 디렉토리 경로
  • gpgkey=$value패키지 검증, 공개키 파일 경로
  • gpgcheck=$value패키지 변조 확인 플래그
  • enabled=$value활성화 여부
  • mirrorlist=$valuebaseurl 대신 가까운 서버 찾도록 하는 여부

rpm 서버 구성 시 참고

  • rpm파일이 사용하는 glibc 버전

  • 공유라이브러리 의존성 확인
