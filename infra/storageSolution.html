스토리지 하드웨어 솔루션

스토리지 하드웨어 솔루션 리스트

  • H/W 계층
    • Hardware RAID - 안정성
  • S/W 계층
    • OS 레벨
      • LVM - 유연성
      • Software RAID - 안정성
    • FileSystem 레벨
      • ZFS / Btrfs
    • Infrastructure 레벨
      • SDS(Software Defined Storage) - 제어의 주체
      • HCI(Hyper-Converged Infrastructure) - 물리적 구조
      • Disaggregated Storage, CDI(Composable Disaggregated Infrastructure)
      • 참고 - Fabric은 어디서든 데이터를 끌어올 수 있다는 의미의 네트워크 망을 의미

스토리지 효율화 기술

  • 전송 규격 기술: NVMe-over-Fabrics(NVMe-oF)
  • Erasure Coding - 데이터 보호 알고리즘, 인프라 관점
  • 중복 제거(Deduplication) & 압축(Compression)
  • 스토리지 계층화: 티어링(Tiering) & 캐싱(Caching)

SDS/HCI 관리 콘솔

  • 개별 호스트 접속
  • 중앙 관리 콘솔
  • 하드웨어 관리 콘솔

SDS/HCI 네트워크 구성

  • 관리망
  • 스토리지망(SAN)
  • 서비스망

CDI 관리 콘솔

  • 개별 컴퓨팅 서버 접속
  • 오케스트레이터(중앙 제어 소프트웨어) 접속
  • 하드웨어 관리 콘솔

CDI 네트워크 구성

  • 관리망
  • 스토리지망(SAN)
  • 서비스망

SAN 종류

  • FC-SAN - 광케이블(Fiber Channel)과 전용 FC-스위치
  • IP-SAN - 이더넷 하드웨어, iSCSI, NVMe-oF, SDS 전용 프로토콜 사용
