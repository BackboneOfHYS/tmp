참고지식

  • 서버에서 인치는 표준
  • 인치는 성인 남성 엄지손가락 한 마디에서 유래
  • 1(in) = 2.54(cm)
  • 피트는 성인 남성 발 크기에서 유래
  • 1(ft) = 12(in)
  • 3(ft) = 1(yd)
  • 1760(yd) = 1(mi)
  • 렉의 너비는 19(in)
  • 랙의 유닛(높이)는 1U = 1.75(in)
  • 표준 랙의 전체 높이 = 42U
  • 랙은 수납 공간, 샤시는 장비들의 통합 몸체, 샤시는 랙 안에 장착

저장 매체 종류에 따른 분류

  • HDD(Hard Disk Drive), Spinning Disk - 부품, 자기 디스크, 기계식
  • SSD(Solid State Drive), Solid State - 부품, 반도체 저장 장치, 플래시 메모리, 전자식
  • SCM(Storage Class Memory, 부품, SSD보다 빠르고 주소 지정 가능)
  • All-HDD (ALL-SATA/SAS) - 전부 HDD
  • All-Flash - 전부 SSD
  • Hybrid - HDD와 SSD 모두 활용

외형에 따른 분류

  • Card Type(추가 부품 형태), 얇은 판 모양, PCB에 칩을 박는 형태
  • JBOD(Just a Bunch Of Disks) - 그저 디스크 묶음, 디스크 슬롯만 가득 찬 박스, 연결된 서버에서 데이터 관리
  • Storage Array - 자체 CPU(컨트롤러)와 OS 존재, 자체 관리

연결에 따른 분류

  • DAS(Direct Attached Storage) - 서버 본체에 케이블로 연결
  • NAS
  • SAN

설치 형태에 따른 분류

  • Rack-Mountable - 랙에 고정될 수 있는 형태
  • Tower - 타워형, 일반적인 PC 본체 모델
  • Blade - 커다란 전용 통(샤시) 하나에 얇은 서버들이 꽂힘
  • Rugged/Portable - 외부 강화 프레임, 들고 다닐 수 있는 손잡이
  • LFF(Large Form Factor) - 3.5인치 드라이브
  • SFF(Small Form Factor) - 2.5인치 드라이브`

물리/논리 프로토콜

  • SATA 계열 - SATA 포트(L자형 커넥터), AHCI 프로토콜, HDD, 2.5인치 SSD, 전원-데이터 케이블 분리
  • M.2 포트, 막대형 NVMe SSD
  • SAS 게열 - SAS 포트(서버용), SCSI 프로토콜, 전원-케이블 일체형
  • PCIe 계열 - PCIe 슬롯/레인, NVMe, CXL 프로토콜

물리 프로토콜에서 NIC/HBA 차이

  • UTP 케이블과 NIC의 관계처럼
  • 광 케이블과 HBA가 관계이다
  • 광 케이블의 대역폭 차이는 GBIC/SFP(Mini-GBIC)로 호환할 수 있다.
