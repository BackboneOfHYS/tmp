금융권 인프라 용어 정리 (대화 기준)

채널/계정계 구조, AP·미들웨어, DB 연결/트랜잭션, 배치 관점에서 자주 나오는 단어들을 실무식 한 줄 설명으로 정리.
전형적인 흐름 큰 그림

채널계 → WEB → AP → 미들웨어(Tuxedo/연계) → 계정계(DB)

실시간(Online) 거래와 배치(Batch)가 동시에 존재하며, “돈이 움직이는” 구간은 계정계 중심으로 엄격하게 보호됨.

채널계Front

고객 접점 시스템(모바일/인터넷/ATM/콜센터). 트래픽 많고 UX 중심.

계정계Core Banking

잔액·이체·입출금·대출·이자 등 “돈”을 직접 처리하는 핵심 시스템. 무결성 최우선.

정보계Analytics

조회/통계/리포트 중심. 읽기 위주이며 실시간성 요구가 상대적으로 낮음.

WEB 서버Edge

HTTP 요청 수신, 정적 리소스 처리, 리버스 프록시/게이트 역할. AP 앞단.

AP (Application Server)Logic

비즈니스 로직 실행 및 DB 접근. 트랜잭션 시작/종료 지점이 되는 경우가 많음.

미들웨어Middle

시스템 간 연결/라우팅/장애 격리/표준화 계층. 금융권 인프라의 “완충지대”.

TuxedoTP Monitor

초고성능 트랜잭션 미들웨어. 계정계 중심에서 서비스 분배/트랜잭션/고가용성 관리.

연계 미들웨어(“뉘우스” 계열)Integration

AP ↔ 타시스템/외부기관 연동 표준화. 직접 호출을 막고, 라우팅·로그·격리 제공.

※ 회사/현장에 따라 이름이 NEWS/NUS 등으로 다르게 불릴 수 있음(사내 명칭인 경우 많음).

트랜잭션ACID

작업의 최소 단위. “전부 성공/전부 실패” 보장(계정계에서 특히 중요).

커넥션DB Link

AP ↔ DB 연결 1개. 생성/인증 비용이 크고 자원 제한이 존재.

커넥션 풀Pool

미리 만든 커넥션을 재사용하는 구조. 성능/안정성 핵심이며 고갈 시 장애로 직결.

커넥션 릭(Connection Leak)Incident

커넥션이 반납되지 않아 풀이 점점 고갈되는 상태. 타임아웃/서비스 멈춤 원인.

External Procedure CallDB→OS

DB/PLSQL이 외부 프로그램(C/C++ 등)을 호출. 가능하지만 보안·안정성 리스크가 큼.

배치(Batch)Scheduled

정해진 시간/조건에 대량 처리(이자·정산·마감). 계정계에서 필수 구성.

실시간(Online)Realtime

고객 요청에 즉시 처리(이체/출금/승인). 응답시간과 동시성 관리가 핵심.

EOD / BODDay Cycle

EOD: 하루 마감 배치(정산/확정). BOD: 업무 시작 전 초기화/기준 데이터 준비.

장애 격리Resilience

한 구간 장애가 전체로 번지는 걸 차단. 미들웨어/풀/트랜잭션 설계의 중요한 목표.

M1metricmesurement

(준)실시간 데이터, 원천 트랜잭션 데이터, 계좌별 거래별 로우 데이터

M2metricmesurement

가공, 집계 데이터

프로세스 카운터CPU PC프로세스 수

CPU의 Program Counter 또는 프로세스 수

워드 카운트word count

단어 수

아카이브 파일archive

여러 파일을 하나로 묶은 파일

PDPhysical Dist

실제 서버에 장착된 디스크

SMTSimultaneous Multi-Threading

CPU 1코어에서 두 개 이상의 스레드를 처리하는 기술, Windows의 Hyper-Threading과 같은 개념

TPSTransactions Per Second

초당 트랜잭션 처리량

TRTransactionTransaction Rate

트랜잭션 또는 단위 시간당 트랜잭션 수

런 큐Run Queue

CPU 줄 서 있는 프로세스 수

glibcGNU C Library

리눅스 프로그램이 사용하는 기본 C 표준 라이브러리

매트릭스matrix

시스템 상태 수치 지표

AD 서버Active Directory 서버

마이크로소프트(Windows Server)의 중앙 인증·권한 관리 시스템

LDAPLight Directory Access Protocol

가벼운 계정/정보 디렉터리에 접근하기 위한 통신 규약

sambaSMB

SMB 프로토콜을 리눅스에서 쓰게 해주는 소프트웨어 이름

SMBServer Message Block

네트워크에 폴더가 항상 열려 있음, 여러 사람이 동시에 접근 가능, 실시간 수정/저장, 회사 파일서버 = 거의 이 구조, 주로 Windows

NFSNetwork File System

파일 공유 프로토콜, 주로 UNIX, Linux.

스토리지storage

물리 디스크, RAID, Storage Pool, LUN 생성

스토리지 풀storage pool

다수의 RAID를 하나의 그룹으로 묶은 것

RAIDRedundant Array of Independent Disks

다량의 디스크를 한 번에 합치고 이를 분산저장함. RAID 레벨(뒤 숫자)에 그 유형이 구분됨. 0, 1, 5, 10순으로 striping, mirroring, striping + parity, 1+0

LUNLogical Unit Number

storage pool에서 수도꼭지 느낌. 스토리지 풀에서 외부에 제공하는 논리 블록 장치. 풀 안의 모든 LUN은 RAID 자원 공유

SANtorage Area Network

서버와 스토리지 장치를 고속 네트워크로 연결, 블록 수준 데이터 엑세스 제공, 디스크 공간 자체를 제공

NASNetwork Attached Storage

SAN과는 다르게 디스크 공간이 아닌 파일, 폴더를 제공. 파일시스템을 스토리지에서 직접 구성함

파일시스템file system

파일 포맷이 아닌 디스크의 폴더, 파일 간 구조를 규정함

VFSVirtual File System

리눅스 커널 안의 계층으로 파일시스템의 통합 인터페이스

VDIVirtual Disk Image

가상 머신의 하드 디스크로 사용하는 가상화 파일 포맷

LVMLogical Volume Manager

스토리지 가상화 기술로 물리 디스크를 추상화하여 논리 불륨으로 유연하게 관리하는 리눅스 기술, 커널 2.4버전부터 default.

PVPhysical Volume

LVM에서의 물리 볼륨

VGVolume Group

LVM에서의 볼륨 그룹으로 하나의 VG는 다수의 PV를 하나로 본다.

LVLogical Volume

VG 안의 전체 공간에서 일부를 할당받는다.

AIXAdvanced Interactive eXecute

IBM이 만든 서버용 유닉스 OS, IBM 파워 전용

EDREndpoint Detection & Response

보안위협 탐지, 대응 솔루션, 백신보다 강함

Micro-Segmentation

내부망 통신 최소 허용, 통신보안전략

SCASoftware Composition Analysis

오픈소스 라이브러리 취약점 분석 도구, Synk, Black Duck, OWASP Dependency-Check 등이 있음

iLOIntegrated Lights-Out

원격 서버 관리 기능, HP 서버에 탑재되는 관리 칩셋

iDRACIntegrated Dell Remote Access Controller

Dell 서버 내장된 원격 관리 전용 컨트롤러

DMZ

외부망과 내부망 사이 위치한 중간 네트워크 구역

FEPFront End Processor

대외 채널(ATM, 카드사 등)에서 들어오는 거래를 1차로 처리하는 중게서버

DDPData Detection & Protection

데이터 유출 탐지 & 보호 솔루션

RVToolsRob de Veij, 개발자명

VMWare 접속 유틸리티

EMC

스토리지 전문 회사, 현재 DELL에 인수

Veritas

백업, 복구, HA 전문 SW 기업

OMSAOpenManage Server Administrator

Dell 서버 내부 하드웨어 상태 관리/모니터링

grafana

서버 데이터 시각화 SW

VM hang

가상머신이 멈춰서 응답이 없는 상태

ping down

ICMP 응답이 안 되는 상태

HBAHost Bus Adapter

서버를 스토리지 네트워크(SAN)에 연결해주는 어댑터 카드

FCFibre Channel

SAN 네트워크 유형

빠른 암기 요약

WEB은 “받고”, AP는 “처리하고”, 미들웨어는 “연결·격리하고”, 계정계는 “돈을 확정”한다.
