가상 메모리(Virtual Memory) 점검 명령어

1) 전체 메모리 사용량 확인

free

free -h
  • Total / Used / Free
  • buff/cache
  • Swap 사용량

2) 실시간 가상 메모리 상태

vmstat

vmstat 1
  • si → swap in
  • so → swap out
  • r → run queue
  • b → block(I/O wait)

3) 프로세스별 메모리 사용량

top

top
  • RES → 실제 물리 메모리
  • VIRT → 가상 메모리
  • SHR → 공유 메모리

ps

ps -eo pid,comm,%mem,vsz,rss --sort=-rss
  • VSZ → 가상 메모리 크기
  • RSS → 실제 메모리 사용량

4) 상세 메모리 정보

/proc/meminfo

cat /proc/meminfo
  • MemTotal
  • MemFree
  • Buffers
  • Cached
  • SwapTotal / SwapFree

5) 특정 프로세스 메모리 맵 확인

cat /proc/PID/status
cat /proc/PID/smaps
  • VmSize → 가상 메모리
  • VmRSS → 실제 메모리

6) OOM(Out Of Memory) 발생 여부 확인

dmesg | grep -i oom
journalctl -k | grep -i oom

7) 스왑 설정 확인

swapon -s
cat /proc/swaps

가상 메모리 핵심 체크 포인트

항목 확인 포인트
free -h Swap 사용 여부
vmstat si/so 증가 여부
top RES 과다 프로세스
dmesg OOM Kill 기록
