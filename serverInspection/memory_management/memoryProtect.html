메모리 보호(Memory Protection) 점검 명령어

1) 프로세스 메모리 권한 확인

pmap

pmap -x PID
  • 메모리 영역별 권한(rwx)
  • Private / Shared 구분

/proc 메모리 맵

cat /proc/PID/maps
  • r → read
  • w → write
  • x → execute
  • p → private
  • s → shared

2) ASLR(Address Space Layout Randomization) 확인

cat /proc/sys/kernel/randomize_va_space
  • 0 → 비활성화
  • 1 → 일부 랜덤화
  • 2 → 전체 랜덤화(권장)

3) NX(Execute Disable) 비트 확인

grep nx /proc/cpuinfo
  • nx 표시 → 실행 불가 영역 지원

4) SELinux 상태 확인

sestatus
getenforce
  • Enforcing / Permissive / Disabled

5) 메모리 잠금(Memory Lock) 확인

ulimit -l
  • mlock 가능 메모리 크기

6) 커널 OOM 보호 설정 확인

cat /proc/PID/oom_score
cat /proc/PID/oom_score_adj
  • OOM 발생 시 종료 우선순위

7) 커널 보호 관련 설정

sysctl kernel.kptr_restrict
sysctl kernel.dmesg_restrict
sysctl vm.mmap_min_addr

핵심 점검 포인트

항목 확인 내용
ASLR randomize_va_space 값
NX 지원 cpuinfo에 nx 존재 여부
메모리 권한 /proc/PID/maps 권한 확인
SELinux Enforcing 상태 여부
OOM 설정 oom_score_adj 값
