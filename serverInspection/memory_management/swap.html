스왑(Swap) 관련 점검 명령어

1) 전체 스왑 사용량 확인

free

free -h
  • Swap Total / Used / Free 확인

swapon

swapon -s
swapon --show
  • 활성화된 스왑 장치 확인

2) 실시간 스왑 활동 확인

vmstat

vmstat 1
  • si → swap in (디스크 → 메모리)
  • so → swap out (메모리 → 디스크)

3) 페이지 단위 스왑 통계

sar

sar -W 1 5
  • pswpin/s → 초당 swap in
  • pswpout/s → 초당 swap out

4) /proc에서 직접 확인

cat /proc/swaps
cat /proc/meminfo | grep -i swap
  • SwapTotal
  • SwapFree
  • SwapCached

5) 특정 프로세스 스왑 사용량 확인

smem (설치 시)

smem -rs swap

/proc

cat /proc/PID/status | grep VmSwap

6) 스왑 관련 커널 설정 확인

sysctl vm.swappiness
  • 0~100 (기본값 보통 60)

핵심 점검 포인트

항목 확인 내용
Swap 사용량 free -h로 Used 증가 여부
Swap 활동 vmstat의 si/so 증가 여부
프로세스별 사용 VmSwap 값 확인
Swappiness vm.swappiness 설정 값
