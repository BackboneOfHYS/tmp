프로세스 생성 및 종료 체크 명령어 (Linux)

1) 현재 실행 중인 프로세스 확인

전체 프로세스 보기

ps -ef

트리 구조 확인

pstree -p

✔ 부모-자식 관계 확인 가능
✔ 특정 서비스가 어떤 프로세스를 생성했는지 파악

2) 실시간 프로세스 생성/종료 확인

top

top

htop (설치 시)

htop

✔ 새로 생성되는 프로세스 관찰
✔ 종료되는 프로세스 확인

3) 특정 프로세스 생성 확인

프로세스 검색

pgrep nginx
ps -ef | grep nginx

✔ 서비스 시작 시 프로세스가 정상 생성되었는지 확인

4) 프로세스 상태 확인

ps -eo pid,ppid,cmd,state
State 의미
RRunning (실행 중)
SSleeping (대기 중)
DI/O 대기
ZZombie (종료 후 부모가 수거 안 함)
T정지 상태

5) 좀비 프로세스 확인

ps -ef | grep defunct

✔ 좀비(Z) 상태 프로세스가 많으면 부모 프로세스 문제 의심

6) 프로세스 종료

정상 종료 요청

kill PID

강제 종료

kill -9 PID

이름으로 종료

pkill nginx
killall nginx

7) 프로세스 생성 로그 확인

journalctl -xe
dmesg

✔ OOM으로 프로세스 종료되었는지 확인 가능
✔ 서비스 재시작 기록 확인

면접용 한 줄

프로세스 생성 및 종료는 ps, pstree, pgrep, top 등을 통해 확인하며, 상태(State) 값과 좀비 여부를 함께 점검합니다.
