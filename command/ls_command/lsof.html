Linux lsof 명령어 정리

lsof는 기본 명령어가 아님

lsof 옵션

  • lsof [경로]: 기본 조회
  • lsof +D [경로]: 디렉토리 내의 파일을 연 프로세스 조회
  • lsof -u [사용자명]: 사용자명 기준 조회
  • lsof -u ^[사용자명]: 특정 사용자 제외한 모든 사용자가 열고 있는 파일
  • lsof -i: 네트워크 연결 확인
  • lsof -i :[포트번호]: 특정 포트를 사용중인 프로세스
  • lsof -i [tcp|udp]: tcp 또는 udp 연결 확인
  • lsof -p [PID]: PID가 열고 있는 파일
  • lsof -a: 옵션의 모든 것을 and조건으로 바꿈, 기본은 or조건임
