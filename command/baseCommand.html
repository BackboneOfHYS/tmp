Linux 기본 명령어

참고사항

  • /dev/null — 들어오는 모든 입력 버림

권한 설정

  • SUID/SGID — 실행시 파일 소유자/그룹 권한으로 실행, 설정되면 x대신 s, x가 없는데 s있으면 S
  • Sticky bit — 디렉토리에서 삭제 제한, 디렉토리에 w권한있으면 파일삭제 가능(파일 권한과 무관)
  • useradd [계정명] — 유저 생성(리눅스)
  • userdel [계정명] — 유저 삭제(리눅스)
  • groupadd [계정명] — 그룹 생성(리눅스)
    • -g그룹 아이디 지정
  • groupdel [계정명] — 그룹 삭제(리눅스)
  • mkuser home=?, pgrp=?, groups=?, — 유저 생성(유닉스)
  • passswd 계정명 — 유저 비밀번호 설정
  • gpasswd [그룹명]— 그룹 비밀번호 설정
    • -a 유저명유저 추가
    • -d 유저명유저 삭제
    • -M 유저1, ...그룹멤버 세팅
    • -A 유저1, ...관리자 세팅
    • -r비밀번호 삭제
    • -Rnewgrp 명령어와 비밀번호를 통한 그룹 접근 금지 설정
    • -R/etc/gshadow에 !설정이면 -R옵션 적용임
    • -R해당 옵션 취소하려면 그룹 비밀번호 재설정
    • -Q/etc가 아닌 다른 마운트포인트를 가리킴
  • chmod 권한 파일명 — 파일 권한속성 변경
  • chown 소유자:소유그룹 파일명 — 파일 소유자, 소유그룹 변경
  • usermod — 계정 설정(리눅스)
    • -g — 기본 그룹 변경
    • -G — 보조 그룹 변경
    • -a — 보조 그룹 변경시 기존 그룹 유지
    • -d — 홈 디렉토리 변경
    • -m — -d 사용시 실제 디렉토리 내용을 같이 이동
    • -s — 로그인 쉘 변경
    • -L — 계정 잠금
    • -U — 계정 잠금 해제
    • -U — 계정 잠금 해제
    • -l — 사용자 이름 변경
  • chuser — 계정 설정(유닉스)
  • groupmod— 그룹 설정(리눅스)
    • -a 유저명그룹에 유저 추가

copy

  • -i — 대화형 확인
  • -p — 속성보존
  • -r — 디렉토리 전체 복사
  • -f — 강제 복사, 읽기 전용 파일도 강제로 덮어씀
  • -v — 상세 출력

grep

  • -i — 대소문자 구분 안 함
  • -c — 매칭된 라인 수만 숫자로 출력
  • -e — 해당 옵션으로 검색어 다수 지정 가능 -e 패턴 -e 패턴, 하이푼을 패턴에 넣고 싶을 때도 활용
  • -v — 해당 패턴이 없는, 반대로 출력
  • -E 정규식egrep과 동일
  • grep -r "패턴" 경로1 경로 2 ... — 지정 경로내의 모든 파일에서 패턴 찾기

stat 경로

  • -c "포맷문자열" — 포맷 지정 출력
  • %n — 파일 이름
  • %s — 파일 크기(bytes)
  • %U — 소유자 이름
  • %u — UID
  • %G — 그룹 이름
  • %g — GID
  • %a — 권한 (숫자)
  • %A — 권한 (문자)
  • %F — 파일 타입
  • %i — inode 번호
  • %h — 하드링크 개수
  • %x — 접근 시간
  • %y — 수정 시간
  • %z — 상태 변경 시간
  • -L — 심볼릭 링크에서 원본파일 정보
  • -f — 주어진 경로가 속한 슈퍼블록(파일시스템)의 정보 출력

ls

  • -t — 시간순 정렬
  • -r — 정렬 역으로 적용
  • -h — 파일 크기 huaan readable
  • -S — 파일 크기순 정렬

date

  • date '+포맷' — 출력 형식 지정에서 '+'는 필수

find

  • -type — 파일 타입, f ,d, l 등
  • -maxdepth — 탐색 제한 설정
  • -printf — 출력 형식 제한, "%f" <- 파일명만
  • -find . -name "이름" — 현재 위치에서 하위까지 탐색하여 일치한 것 찾음

tar

  • -v — 출력 보기
  • -f — 파일명 지정
  • -t — 내용물 보기
  • -c — 새로 묶기
  • -x — 압축 풀기
  • -z — 단순 압축

gzip

  • gzip — gzip은 기존 파일 삭제, 하나의 파일만 압축하기 때문에 tar로 묶고 압축
  • -1~9 — 압축 속도, 압축률 조정
  • -v — 실행처리 과정 출력
  • -r — 디렉터리 내부 재귀 압축
  • -d — 압축 해제
  • -c — 리다이렉션으로 저장

리다이렉션, here string

  • 파일 vs 명령어
  • >>> — 문자열을 표준입력(stdin, FD 0)으로 직접 전달 기능
  • >> — 덮어쓰기
  • > — 이어쓰기

파이프라인

  • 명령어 vs 명령어
  • | — 기본 문법

sort

  • -h — human readable기준 정렬
  • -r — 옵션키면 내림차순, 안 키면 오름차순
  • -k n — n번째 컬럼 기준

백그라운드 작업관리

  • & — 명령어 제일 뒤에 추가하면 백그라운드로 실행
  • jobs — 백그라운드로 실행중인 프로세스 출력
  • jobs로 출력되는 색인은 fg, bg에서 %로 접근가능
  • fg %n — 백그라운드 프로세스를 포그라운드로 전환
  • bg %n — 포그라운든 프로세스를 백그라운드로 전환
  • kill %n — 백그라운드 작업 종료
  • nohup — 터미널이 종료되어도 계속 실행
  • setsid — 세션 완전 분리

read

  • -r — 이스케이프 무시

logname / whoami

  • 세션 첫 로그인 기준 / 현재 기준
  • who는 현재 기준

uname

  • 시스템 정보 출력
  • -a — 전부 출력
  • -m — CPU 아키텍처
  • -n — 네트워크 호스트 이름
  • -r — 커널 릴리즈 버전
  • -s — 운영체제 커널 이름
  • -v — 커널 빌드 정보
  • -p — 프로세서
  • -i — 하드웨어 플랫폼
  • -o — os 이름

arch

  • CPU 정보 확인

env

  • 환경변수 출력

which

  • 명령어 위치 출력

history / !n

  • 사용했던 명령어 기록 확인 및 n번째로 실행한 명령어 실행
  • -d n/ -c — n번째 명령어 삭제, history 전체 삭제

wc

  • wc 파일경로 -> "행 단어 비트 파일명" 출력
  • -l — 행 수 출력
  • -w — 단어 수 출력
  • -c — 바이트 수 출력
  • -m — 글자 수(멀티바이트 처리 유리)
  • -L — 가장 긴 줄의 길이가 몇 글자인지

tr

  • 문자 치환 명령어
  • tr 'a-z' 'A-Z' — 소문자->대문자
  • -d 'a'a 문자 제거
  • -s 연속으로 반복되는 문자를 하나로 압축
  • -c 'a'지정 문자 세트를 제외한 나머지 전체를 대상
  • -t 첫 번째 세트의 길이를 두 번째 세트 길이에 맞게 자름
