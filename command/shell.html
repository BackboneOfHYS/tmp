Linux bash shell 기본 문법

참고지식

  • FD — 프로세스에서 접근(열기)한 모든 파일에 대한 색인, 0, 1, 2는 각각 stdin, stdout, stderr가 기본
  • glob vs 정규식 — 정규식은 패턴매칭, glob(와일드카드)는 파일명 매칭

입출력 및 기본

  • read -r 변수명 — '한 줄' 입력, -r옵션은 \를 이스케이프로 인식하지 않음
  • read -d '구분자' — 구분자를 만나기 전까지 읽음
  • echo/prinf "$변수명" —
  • sleep n — sleep 명령어
  • $RANDOM — 난수
  • 변수명=$(명령어) — 명령어 결과 저장

변수

  • "$var" — 기본 사용 문법
  • var=10 — 기본 할당 문법
  • 모든 변수의 기본은 문자열로 가정한다.
  • 0b... — 2진수 취급
  • 10진수는 프리픽스 없는 것
  • 0n... — 8진수 취급
  • 0xn... — 16진수 취급
  • n#mn진법 m

조건문

  • if [ 변수 = 변수 ]; then — 기본 문법
  • elif [ 변수 = 변수 ]; then — 기본 문법
  • else — 기본 문법
  • fi — 기본 문법
  • [[ 변수 =~ 정규식 ]]; — 정규식 조건
  • (( 변수 -eq 변수 )); — 산술연산영역, 조건 아님!

while

  • while [ ] — 기본 문법
  • do — 기본 문법
  • done — 기본 문법
  • while true — 무한 루프
  • while IFS= read -r line — 파일 읽기
  • done < [파일명] — 파일 읽기
  • while read file — 명령어 결과 읽기
  • done < <(ls) — 명령어 결과 읽기

명령어 연결자

  • | — 파이프라인
  • && — 전방 명령 성공시 후방 명령 실행
  • ; — 전방 명령 성공 유무 무관하게 후방 명령 실행
  • || — 전방 명령 실패시 후방 명령 실행

쉘 특수 변수

  • $? — 직전 명령 종료 코드
  • $$ — 현재 쉘 PID
  • $! — 마지막 백그라운드 PID
  • $0 — 스크립트 이름
  • $1~$n — 위치 인자
  • $# — 인자 개수
  • $@ — 전체 인자(권장)
  • $* — 전체 인자
  • $UID — 사용자 ID
  • $EUID — 유효 사용자 ID
  • $PPID — 유효 사용자 ID
  • $PWD — 현재 디렉토리
  • $IFS — 필드 구분자

쉘 확장 순서

  • {1...10} — 단순 문자열 생성기
  • ~ — 홈 디렉토리 확장
  • ${var} — 변수 확장
  • $(command) — 명령 실행 결과 반영
  • $((정수연산)) — 정수 계산
  • "a b c" — IFS 기준 분리
  • * — 파일 확장 패턴, 와일드카드, glob
