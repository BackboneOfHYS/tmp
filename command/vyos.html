vyos concept

기본 출력 명령어

  • show interfaces: 인터페이스 상태 및 IP 확인
  • show configuration: 현재 실행 중인 전체 설정 보기
  • show configuration nat: NAT 설정 확인
  • show configuration firewall
  • show configuration commands firewall: 방화벽 규칙 트리 구조로 보기
  • show nat source translations: 실시간 변환 현황 확인
  • show nat destination translations: 실시간 변환 현황 확인
  • show nat source statistics: 규칙별 통계 확인
  • show firewall name <정책이름> statistics: 특정 방화벽 장책 상세 통계 확인
  • show firewall summary: 요약된 방화벽 정보 확인
  • show route: 라우팅 테이블 확인
  • show log: 시스템 로그 확인
  • monitor interface [이름]: 특정 인터페이스 트래픽 실시간 모니터링

설정 모드 (Configuration Mode)

  • set interfaces ethernet eth0 address [IP/마스크]: IP 주소 할당
  • set protocols static route [네트워크] next-hop [게이트웨이]: 정적 라우팅 설정
  • set service ssh port 22: SSH 서비스 활성화
  • set system host-name [이름]: 호스트 이름 설정
  • set system login user [ID] authentication plaintext-password [비번]: 사용자 계정 생성
  • delete [경로]: 특정 설정 항목 삭제

관리 및 도구

  • reboot: 시스템 재부팅
  • poweroff: 시스템 종료
  • ping [IP]: 네트워크 연결 확인
  • traceroute [IP]: 경로 추적
  • show version: VyOS 버전 및 하드웨어 정보 확인

방화벽

  • set firewall name [이름] default-action [drop|accept]: 기본 정책 설정
  • set firewall name [이름] rule [번호] action [accept|drop]: 특정 규칙의 동작 지정
  • set firewall name [이름] rule [번호] destination port [포트]: 목적지 포트 지정
  • set firewall name [이름] rule [번호] protocol [tcp|udp|icmp]: 프로토콜 지정
  • set firewall name [이름] rule [번호] source address [IP]: 출발지 주소 지정
  • set interfaces ethernet eth0 firewall [in|out|local] name [이름]: 인터페이스에 방화벽 적용

NAT

  • NAT 설정에서 rule은 정책(Rule Set)의 rule과 다름
    • set nat source rule [번호] outbound-interface [인터페이스]: 외부로 나가는 통로 지정
    • set nat source rule [번호] source address [내부IP범위]: 변환할 내부 주소 범위
    • set nat source rule [번호] translation address masquerade: 외부 인터페이스 IP로 자동 변환
    • set nat destination rule [번호] inbound-interface [인터페이스]: 들어오는 통로 지정
    • set nat destination rule [번호] destination port [외부포트]: 외부에서 접속할 포트
    • set nat destination rule [번호] protocol [tcp|udp]: 프로토콜 지정
    • set nat destination rule [번호] translation address [내부서버IP]: 전달할 내부 서버 IP
