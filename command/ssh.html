Linux ssh 명령어 정리

passphrase

  • 개인키를 다시 암호화하는 용도

ssh-agent

  • -s — sh/bash 문법으로 환경변수 설정 코드 출력
  • -c — csh/tcsh 문법으로 환경변수 설정 코드 출력
  • eval "$(ssh-agent -s)" — ssh-agent 환경변수 적용
  • -k — 메모리에서 키 제거됨

ssh-add

  • ssh-add ~/.ssh/id-ed25519 — ssh 개인키 등록
  • -I — 등록된 키 목록 확인
  • -L — 등록된 키의 공개키 출력
  • -t — 키 유효 시간 설정(Time)
  • -x — 에이전트 잠금
  • -X — 에이전트 잠금 해제
  • -d 키경로 — 키 하나만 제거
  • -D — 키 전체 제거

ssh [옵션] 유저명@주소

  • -p — 포트 지정
  • -i — 개인키 지정
  • -v, -vv, -vvv — 디버그 모드, 단계별로 자세한 출력
  • -L — 로컬포트 포워딩
  • -R — 리버스 포트 포워딩
  • ssh -L(R) port:host:port 유저명@주소 —
  • -T — tty 할당 안 함

scp

  • -r — 디렉토리 전체 복사
  • -P — 포트 지정
  • -i — 개인키 지정
  • -p — 원본 파일 속성 유지
  • -C — 압축해서 전송
  • -l — 대역폭 제한
  • scp 파일명 유저@호스트:경로 — 로컬->원격
  • scp 유저@호스트:경로 경로 — 원격->로컬
  • scp 유저@호스트:경로 유저@호스트:경로 — 원격->원격

ssh-keygen

  • ~/.ssh/ — 저장 경로
  • ~/.ssh/id_타입명 — 개인키
  • ~/.ssh/id_타입명.pub — 공개키
  • -t — 타입 지정(rsa, ed25519)
  • -C — 주석
  • -b — 키 길이 지정

ssh-copy-id 유저명@호스트명

  • ~/.ssh/authorized_keys — 공개키 저장 파일
