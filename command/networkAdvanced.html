Oracle Linux (RHEL 계열) 네트워크 명령어 리스트

“네트워크 확인/트러블슈팅 때 바로 쓰는 명령어”만 간단히 정리

리눅스 네트워크 내부 구조

  • User Command — 사용자 명령어
  • libnl — 유저공간 라이브러리
  • netlink socket (IPC) — IPC
  • Kernel netlink handlers — 커널 핸들러
  • Kernel Network Tables — 커널 네트워크 테이블

간단 용어 정리

  • family리눅스 네트워크 서브시스템의 분류 기준
  • IPCInter-Process Communication, 프로세스 간 통신
  • D-Bus유저 공간 프로그램끼리 말하게 해주는 메시지 버스
  • libnmD-Bus로 NetworkManager와 D-Bus로 쉽게 통신하도록 해주는 API

User Command

  • ip
  • ss
  • tc
  • bridge

libnl

netlink를 쓰기 쉽게 만든 사용자 공간 라이브러리
nmcli로 Networkmanager를 다루고 NetworkManager가 이 계층을 다룸

netlink

유저와 커널 사이 네트워크 전용 IPC
ip 명령어는 netlink를 직접 사용
ss 명령어는 netlink를 직접 사용

Kernel netlink handlers

family별로 handler 함수가 등록되므로 family별로 함수 분기 진행

Kernel Network Tables

netlink를 통해서만 조회/수정 가능한 네트워크 설정 테이블
