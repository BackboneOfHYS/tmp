Linux 프로세스 기본 정리

top

  • P — CPU 기준 정렬
  • M — 메모리 기준 정렬
  • T — 누적 CPU 사용 시간 기준 정렬
  • < — 오름차순 정렬
  • > — 내림차순 정렬
  • R — 현재 기준 역순 정렬
  • u — 유저 기준 검색
  • k — 프로세스 종료
  • r — pid 우선순위 번경
  • / — 검색(불확실)
  • o — 컬럼명=값 으로 필터링
  • = — o 옵션 취소
  • c — COMMAND의 전체 경로와 옵션 출력
  • L — 전체 목록에서 하이라이트하기
  • 1 — CPU 코어별로 사용률 보여줌
  • E — 상단 수치 단위 변경
  • e — 하단 프로세스 수치 단위 변경
  • m — 상단 메모리 사용량 부분을 그래프로 바꾸거나 끔
  • t — 상단 CPU 사용량 부분을 그래프로 바꾸거나 끔
  • b — 현재 실행중인 프로세스를 굵게 표시하거나 색을 반전
  • X — 현재 정렬 기준 컬럼 하이라이트
  • d — 갱신 주기 변경
  • W — 현재 화면 구성 설정 영구 저장

ps

  • 유닉스, BSD 스타일 등으로 나뉨
  • -e — 모든 프로세스 출력, -A와 동일
  • -f — 상세 정보 풀 포맷으로 출력
  • -a — 터미널과 연관된 모든 사용자 프로세스
  • -u [사용자] — 사용자 중심의 상세 출력
  • -x — 터미널이 없는(데몬 등) 프로세스까지 포함
  • -C [명령어] — 실행 파일의 이름을 기준으로 검색
  • -p [PID] — 특정 PID를 가진 프로세스 출력
  • -F — -f보다 더 상세한 정보 출력
  • -l — 긴 포맷으로 출력
  • -o [출력항목] — 출력할 항목 직접 지정
  • --sort=-mem — mem 사용량 기준 정렬, -가 있으므로 역정렬
  • 출력항목
    • UID / USER — 프로세스를 실행한 사용자 계정
    • PID — 프로세스 고유 ID
    • PPID — 부모 프로세스 ID
    • %CPU/%MEM — CPU 및 물리적 메모리 백분 사용률
    • VSZ / RSS — 가상 메모리 / 실제 사용 중인 메모리 크기(KB)
    • STAT — 프로세스 상태
    • STIME / START — 프로세스가 시작된 시간
    • TIME — 총 CPU 사용 시간
    • COMMAND — 실행된 명령어 이름 또는 경로

pgrep

  • -f — Full Command Line, 실행 시 사용한 전체 경로와 인자까지 포함하여 검색
  • -u — 특정 사용자가 실행한 프로세스
  • -g — 특정 그룹이 실행한 프로세스
  • -t — 터미널에서 실행중인 프로세스
  • -x — 패턴이 이름과 정확히 일치할 때
  • -l — PID와 프로세스 이름 출력
  • -a — PID와 함께 전체 실행 경로 및 인자 출력
  • -c — 조건에 맞는 프로세스 개수만 출력
  • -n — 가장 최근 실행된 것
  • -o — 가장 오래된 것

kill / pkill

  • kill [PID] — PID로 프로세스 종료
  • pkill [옵션] — pgrep과 옵션 거의 동일
