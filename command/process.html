Linux 프로세스 기본 정리

top

  • P — CPU 기준 정렬
  • M — 메모리 기준 정렬
  • T — 누적 CPU 사용 시간 기준 정렬
  • < — 오름차순 정렬
  • > — 내림차순 정렬
  • R — 현재 기준 역순 정렬
  • u — 유저 기준 검색
  • k — 프로세스 종료
  • r — pid 우선순위 번경
  • / — 검색(불확실)
  • o — 컬럼명=값 으로 필터링
  • = — o 옵션 취소
  • c — COMMAND의 전체 경로와 옵션 출력
  • L — 전체 목록에서 하이라이트하기
  • 1 — CPU 코어별로 사용률 보여줌
  • E — 상단 수치 단위 변경
  • e — 하단 프로세스 수치 단위 변경
  • m — 상단 메모리 사용량 부분을 그래프로 바꾸거나 끔
  • t — 상단 CPU 사용량 부분을 그래프로 바꾸거나 끔
  • b — 현재 실행중인 프로세스를 굵게 표시하거나 색을 반전
  • X — 현재 정렬 기준 컬럼 하이라이트
  • d — 갱신 주기 변경
  • W — 현재 화면 구성 설정 영구 저장

ps

  • -e — 모든 프로세스 출력, -A와 동일
  • -f — 상세 정보 풀 포맷으로 출력
