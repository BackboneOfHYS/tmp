I/O Controller

참고지식

  • PIIX, ICH - 인텔 구형 메인보드 칩셋 이름
  • IDE(Integrated Drive Electronics) - 데이터 전송 방식(프로토콜), PATA 방식
  • AHCI(Advanced Host Controller Interface) - 데이터 전송 방식(프로토콜), SATA 방식의 성능 최고 향상
  • 컨트롤러는 host, disk 양쪽에 존재하며 이 사이를 ATA, SCSI, NVMe 등의 프로토콜(규격)이 통신을 보장한다
